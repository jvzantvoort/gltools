# glt init

Initialize the local gitlab config.

## Synopsis

```
glt groups [-g|--gitlab <gitlabsection>] [-q|--quiet] [-v|--verbose]
```

## Description

**glt init**

## Options

- `-g, --gitlab GITLABSECTION`

  Which configuration section should be used (default: local)

- `-q, --quiet`

  Silence warnings

- `-v, --verbose`

  Enable verbose output
